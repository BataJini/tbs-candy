# 🍬 TBS CANDY

**A first-person open-world game set in a surreal version of Tbilisi.**

Tag walls, deliver fictional candy, drive taxis, and get lost in the city — right in your browser.

## ✨ Features

- 🏙️ Open-world Tbilisi — 5 connected zones, a valley ringed by mountains and split by the river
- 🎨 Spray graffiti on 45 spots, deliver candy, meet weird NPCs, and unlock 3 endings
- 🚕 Drive taxis, cars and bikes around a living city with traffic and couriers
- 🌗 Day/night cycle, drifting clouds, and a Fallout-style radar minimap
- 🎧 100% procedural art and audio — generated at runtime, no external assets

## 🎮 Controls

| Key | Action |
| --- | --- |
| `WASD` | Move |
| `Mouse` | Look |
| `Space` | Jump |
| `Shift` | Sprint |
| `C` | Crouch |
| `E` | Talk / interact · **hold** to spray · enter/exit car |
| `WASD` *(in car)* | Drive · `Space` = handbrake |
| `F` | Camera |
| `Tab` | Menu (map, tags, candy, people, settings) |
